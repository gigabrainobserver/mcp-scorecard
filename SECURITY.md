# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in MCP Scorecard, please report it responsibly.

**Email:** security@mcp-scorecard.ai

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 48 hours and aim to provide a fix within 7 days for critical issues.

## Scope

This policy covers:
- The scoring pipeline (`src/mcp_scorecard/`)
- The API layer (`api-worker/`)
- The MCP server (`src/mcp_scorecard/server.py`)
- The static site served at mcp-scorecard.ai
