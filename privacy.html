Privacy Policy

The short version: We don't collect anything. No data. No cookies. No tracking. Nothing.

What we collect

Nothing. This site is static HTML served from GitHub Pages. We have no server-side code, no database, no analytics, and no way to identify you.

Cookies

This site sets zero cookies. No session cookies, no tracking cookies, no third-party cookies. None.

Analytics and tracking

We use no analytics services. No Google Analytics, no Plausible, no Fathom, no tracking pixels, no fingerprinting. We don't know how many people visit this site, and we don't want to know.

Third-party requests

This site makes no requests to third-party services. All assets (HTML, CSS, JSON data) are served from the same domain. No CDNs, no external fonts, no social media widgets.

Data we publish

MCP Scorecard publishes trust scores derived entirely from publicly available data:

  • Public MCP registry listings
  • Public GitHub repository metadata (stars, forks, license, commit history)
  • Public package registry metadata

We do not collect, store, or publish any private or personal data about server authors or users.

GitHub Pages

This site is hosted on GitHub Pages. GitHub may collect basic server logs (IP addresses) as part of standard web hosting. This is governed by GitHub's Privacy Statement, not by us. We have no access to these logs.
